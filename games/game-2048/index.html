2048 Animated

Choose Easy 6×6 or Hard 4×4, then slide matching numbers to reach 2048.

Keyboard: arrows/WASD, P pause, R restart. Touch buttons and swipes work on mobile.

Mode

Easy 6×6 gives more space and is easier to win. Hard 4×4 is the classic smaller board.

Goal

Slide all tiles in one direction. Equal numbers merge. Win by creating a 2048 tile. Hard mode uses a 4×4 board. Easy mode uses a 6×6 board.
