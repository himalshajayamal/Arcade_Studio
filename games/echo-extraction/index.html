Echo Extraction

Tactical mission shooter: recover the data core, manage gear, and extract alive.

Move with WASD/D-pad. Tap canvas or Action to shoot. Use Alt to reload/sprint. Press E/F near objects to interact.

Mission Rules

  • Goal: Find a keycard, enter the server wing, download the data core, then extract alive.
  • Win reward: Echo Extraction Badge + loot value saved to the leaderboard.
  • Lose: health reaches zero or the raid timer expires.
  • Tactical systems: noise attracts enemies, ammo is limited, loot has carry weight, and AI guards use patrol/chase/attack states.
