## Intelligent game methods

- Adaptive Tower Defense: adaptive wave director, tower targeting, path following, resistant enemy composition.
- Roguelike Dungeon Crawler: procedural generation, turn-based enemy chase AI, loot, exit state.
- Tactical Board AI: minimax with alpha-beta pruning and board evaluation.
- Stealth Escape: finite state machine guards, vision cones, obstruction checks, hidden state.
- Colony Simulator: resource economy, worker scheduling, production chains, emergency depletion logic.

## Compatibility

- Runs without backend services.
- Uses only HTML, CSS, JavaScript, Canvas, Web Audio API, and localStorage.
- No runtime API calls.
- No external media dependency.
- Metadata is embedded in JavaScript for GitHub Pages and local file compatibility.
