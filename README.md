# Arcade Studio — 58-Game Browser Arcade

A client-side HTML/CSS/JavaScript browser arcade designed for GitHub Pages and local `file://` execution. No backend and no runtime `fetch()` calls are required.

## Architecture Rules

- Root-ready deployment: `index.html`, `css/`, `js/`, `games/`, `docs/`, and `.github/` live directly in the repository root.
- Shared systems: StorageManager, AudioEngine, InputManager, BaseGame, and shared CSS are reused across the arcade.
- Each game has its own folder with `index.html`, game-specific CSS, and game-specific JavaScript.
- Scores, achievements, favorites, settings, and stats are saved with `localStorage` with a safe fallback.
- Sound is generated procedurally with Web Audio API and safely starts after user interaction.
- Games are mobile-friendly and support keyboard, mouse, touch, and virtual controls where applicable.

## Upload Rule

Upload the contents of this ZIP directly to the GitHub repository root. Do **not** upload a nested folder around the project.
